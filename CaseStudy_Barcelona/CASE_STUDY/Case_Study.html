Data Preparation

In this section, the needed data preparation steps required for the analysis is performed. Unnecessary chunk codes are omitted in the compiled pdf-file. First of all, the actual allocation of the immigrants in the city is revealed, for which the âimmigrants_by_nationality.csvâ file was used. Due to the lack of data and the impossibility to differentiating them, all types of migrations were considered, including the internal one, even when this group represents 36.32% of all migrations data in 2017.

The data preparation steps for the different variables are as follows :

1. Accidents - The accidents.csv file contains accidents from the year 2017 along with details like neighbourhood, district and time. For our purposes only the number of accidents per district is required. Along with this the immigrant population is obtained from the immigrants_by_nationality.csv file. It contains the number of immigrants per neighbourhood for different years. Only 2017 data was considered for the accidents. Since areas with higher population would have higher accidents, the number of accidents was normalised using the total population of that neighbourhood and accidents per 1000 people were used for each neighbourhood for analysis.

2. Unemployment - The unemployment.csv and population.csv contains the number of unemployed individuals and population at districts and neighborhood level for the years 2013 - 2017 and the immigrant_emigrants_age consists of immigrants data for the years 2015-2017. The registered unemployed data at neighborhood level for the years 2015-17 is extracted and merged with neighborhoodâs population data for the same time period. The resultant table is then merged with immigrants data at neighborhood level, also the unemployment is then normalized by population and consequently used for analysis and visualization.

3. Connectivity - For understanding the connectivity of each neighborhood, the âbus_stop.csvâ file was used. After tidying up missing or mistaken values, and deleting duplicate ones, the final count of bus stops per neighborhood was calculated. For counting the number of bus stop per area, just the âDay bus stopâ and âNight bus stopâ variables were considered and sum up in the âdayandnigth_countâ column, leaving appart the âAirport bus stopâ and âBus stationâ ones because they are just a few observations and do not really impact in the daily connectivity of the hood.

4. Life Expectancy - The life_expectancy.csv file contains life expectancy data for different neighbourhoods of Barcelona. For our purpose, we have taken the mean life expectancy per neighbourhood. To get the mean life expectancy, we have taken the average of life expectancy over the years 2006-2014. We have further taken the average of male and female population for each neighbourhood. To achieve this we have used melting and casting operations on original data table.

Data Analysis

Analysis on number of bus stops,accidents,number of unemployed people and life expectancy per neighbourhood in relation to the immigrant population. For each variable the p value using spearman correlation test is given. The reason spearman correlation is used is because we observed that the immigrant count per neighbourhood does not follow a Gaussian distribution(figure below). For all the tests, the null hypothesis is as follows : H0 : There is no correlation between the variable and the immigrant count.

From the above 4 plots a number of observations can be made. Firstly, the number of accidents and average life expectancy of a district do not have any correlation with the immigrant population. This is confirmed by the correlation tests which return a significant p value and hence we fail to reject the null hypothesis. However, for unemployment and the number of bus stops we do see some correlation. This can be confirmed by the low p values which means we can reject the null hypothesis. For these variables we will now perform further testing to see if we can fit a linear model on them to try and predict the number of immigrants in a given neighbourhood.

For both of the models, the variance of residuals is not constant and hence the resultant models will not be very effective. Nevertheless, on going ahead with the linear models we obtain the following analysis.

From the current data it appears that the complex model2(immigrant count ~ unemployment+bus stops) works better than the simpler models model3(immigrant count ~ bus stops) and model1(immigrant count ~ unemployment). This is ascertained by the p values of the ANOVA test which are 1.62e-15 and 0.02 respectively. Hence we can reject the null hypothesis, which is that the complex model is not better than the models on individuals factors.

Conclusion

From the performed analysis we conclude that unemployment and bus stops(connectivity) do have some correlation with the number of immigrants in a neighbourhood. In the case of unemployment the correlation(-0.207) tells us that immigrants do tend to live in neighbourhoods with better employment numbers. For the bus stops we see a positive correlation(0.463) which makes sense because of ease of transport in well connected neighbourhoods. This would be particularly important for immigrants who rely heavily on public transport as compared to private vehicles.

On verifying the assumptions for linear regression, we noticed that both unemployment and bus stops, strictly conform to them, However, since the variation in residuals was not a lot the linear models for these data points were compared and the ANOVA test showed that a model using both these factors performed better than a simpler model with either one of these factors.

For accidents we see that there is not much correlation, negative or positive, with immigrant population. This could imply that immigrants donât necessarily care about these numbers and are more focused on factors like cost of living and connectivity. Our assumption was that better life expectancy could relate to better health services or higher safety in a neighbourhood. It seems that this is not a contributing factor in the distribution of immigrants in Barcelona.

Some of the shortcomings of our analysis are the lack of data regarding factors like cost of living which would perhaps have had a significant correlation with the immigrants population.
